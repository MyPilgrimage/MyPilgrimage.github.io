### Chirpy Starter
![Gem Version](https://img.shields.io/gem/v/jekyll-theme-chirpy) &nbsp; 
![GitHub license](https://img.shields.io/github/license/cotes2020/chirpy-starter.svg?color=blue) <br/>
Check out the [theme's docs](https://github.com/cotes2020/jekyll-theme-chirpy/wiki).

- Start server
  `./tools/run.sh`
  or
  `bundle exec jekyll s`
- Check core functions
  `bash ./tools/test.sh`
- Rebuild the JS.
  If you changed the files in the _javascript/ directory, you need to rebuild the JS.
  During development, real-time debugging can be performed with the following commands:
  1. Firstly, start a Jekyll server:
  `./tools/run.sh`
  2. And then open a new terminal session and run:
  `npm run watch:js`
